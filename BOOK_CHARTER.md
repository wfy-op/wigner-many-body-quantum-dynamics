# 《维格纳相空间方法与玻色多体量子动力学》书稿章程

## 一句话目标

给定玻色多体系统的初态、哈密顿量和目标观测量，本书使读者能够构造初始 Wigner 分布、生成随机场样本、执行截断维格纳演化、还原量子观测量，并用独立基准和收敛测试判断结论的可信边界。

## 读者与前置知识

- 主要读者：高年级本科生、研究生和初入门的冷原子/量子光学研究者。
- 前置知识：本科量子力学、统计力学、Fourier 分析和基本 Python 数值计算。
- 二次量子化、Bogoliubov 理论和随机抽样在正文中从可用层次重新建立。

## 范围

- 主体：闭合、弱至中等相互作用、具有高占据模式的玻色系统。
- 核心方法：Weyl--Wigner 表象、Moyal 演化、截断维格纳近似（TWA）。
- 贯穿案例：谐振子、Kerr 振子、Bose--Hubbard 二聚体、离散玻色场、线性耦合双分量凝聚体淬火。
- 扩展：开放系统、positive-P、精确对角化、张量网络和动理学方法。

## 明确边界

- 不把 TWA 描述为任意时间、任意占据和任意相互作用下的精确方法。
- 不以系综平均条纹消失单独证明或否定自发对称性破缺。
- 不把仅含局域线性耦合的双分量模型误称为完整的 Raman 自旋轨道耦合模型。
- 严格一维体系中区分长程序、准长程序、有限尺寸序和陷阱诱导结构。
- 缺少参数或独立证据时，给出方法框架和待验证条件，不制造数值结论。

## 每章交付标准

每章至少包含：中心问题、学习目标、核心推导、一个可检查例子、方法边界、小结和习题。涉及数值结论时，还应包含程序入口、参数、收敛或基准证据。

## 每章审核标准

1. 科学性：公式、量纲、对称性和物理表述正确。
2. 论证性：主要结论附近存在推导、引用或数值证据。
3. 一致性：符号、Fourier 约定、排序约定和单位前后一致。
4. 可复现性：数值图来自实际程序，参数和随机性可追踪。
5. 教学性：章节依赖清楚，例子由简单到复杂，习题与目标对应。
6. 排版性：章节可独立参与全书编译，无未解析引用和破版内容。
