## P1：正式冻结前应关闭

### P1-1　一般 BdG 的 Nambu 符号约定跨章冲突

卷首符号表和附录B定义 `w=(u,v)^T`（`frontmatter/notation.tex:44-47`；`appendices/appB_gaussian.tex:104-113`），第15章为了匹配其场展开却定义 `w=(u,-v)^T`，随后构造 `S=[[U,-V^*],[-V,U^*]]`（`chapters/part4/ch15_bogoliubov_quench.tex:174-210`）。第15章内部是自洽的，但读者若按卷首约定组装本征列，会选错下半块符号和粒子--空穴伙伴。

**关闭条件：** 选定唯一权威约定，在同一处并排给出场展开、正能列、粒子--空穴伙伴和完整 `S`；同步卷首、附录B和第15章；增加一个程序级检查，验证 `S^\dagger\Sigma_zS=\Sigma_z`、`S^{-1}=\Sigma_zS^\dagger\Sigma_z` 以及由 `S` 重构原子场协方差。

### P1-2　`N_W` 同时表示两个不同对象

第7章把 `N_W=\sum_j(|\alpha_j|^2-1/2)` 定义为物理数算符的 Weyl 符号（`chapters/part2/ch07_twa_derivation.tex:173-176`）；第12章又把 `\mathcal N_W=\int|\psi|^2` 定义成未扣半量子的对称范数（`chapters/part3/ch12_propagation.tex:101-110`）；附录D更把原始模方和称为 `N_W`，随后再减1（`appendices/appD_exercises.tex:96-100`）。第16、17章则恢复使用扣除基线后的含义。这个歧义正好落在本书最强调的“轨迹变量不等于观测量”处。

**关闭条件：** 保留 `N_W=(\hat N)_W=\sum|\alpha|^2-M/2`；把原始轨迹范数统一改成 `\mathcal N_{\rm sym}`（或其他不含 `W` 的名称），并在卷首符号表给出两者关系。全书检索 `N_W` 后逐处核对。

### P1-3　capstone 排除了密度零模，却仍按完整正则模式空间扣基线

程序把密度 Goldstone 的 `k=0` 随机系数强制置零（`code/ch14_17_capstone/two_component_field_quench.py:68-106`），但粒子数仍对两个分量各减 `N_x/2`，结构因子仍减完整格点基的 `N_x/2`（同文件 `148-161`）。第16章也承认这只是近似数守恒线性化，并非严格固定 `N` 投影（`chapters/part4/ch16_phase_dynamics.tex:147-164`）。因此抽样相空间与排序修正所假定的投影空间并不相同；按当前构造，原始噪声基线比完整两分量正则空间少半个量子，报告的总粒子数至少存在一个明确的半量子记账歧义，而涉及该约化空间的乘积估计器尚未重新推导。

**关闭条件：** 二选一：

- 实现一致的数守恒凝聚模/集体坐标 Wigner 处理；或
- 明确定义排除密度零模的约化投影核，并从该核重新推导总数、相干重叠和结构因子的 Weyl 符号。

修正后须以真空/高斯初态回测原始范数、物理粒子数和相关函数，证明“抽样模式数 = 基线扣除模式数”。不能只无推导地把常数改成 `1/2`。

### P1-4　capstone 只验证了模式代数，没有验证实际抽样协方差与完整性

初态验收只保存 `u^2-v^2=1`、最小能量和最大热占据（`code/ch14_17_capstone/two_component_field_quench.py:115-127`）；最终门禁也只有辛归一、粒子数、时间阶数、相干的轨迹/截止敏感度（同文件 `481-545`）。没有用样本回测 `\langle|b_k|^2\rangle`、反常矩、密度/自旋协方差，也没有完整 paraunitary/projector 重构残差。第11、15章要求的抽样验收尚未真正施加到全书最重要的端到端案例。

同时，步长、128/256轨迹和48/64/80截止比较都只对相干量执行（同文件 `367-382,392-469,503-524`）；自旋结构因子虽有 jackknife 误差带，却没有相同的收敛检查。当前正文已经正确把它降为估计器与不确定度展示，这一表述可以保留；若要把它升级为物理证据，必须补齐门禁。

**关闭条件：** 指标 JSON 增加逐模正常/反常协方差残差、密度/自旋通道协方差残差、被排除模式清单及完整性残差，并设机器可读阈值；对 `S_s(k_1)` 做步长、轨迹数和截止扫描，或明确把它从“已验证主张集合”中移除。

### P1-5　闭合 capstone 缺少淬火后能量守恒门禁

第12章把 Weyl 数与 Weyl 能量并列为闭合场传播的基本不变量（`chapters/part3/ch12_propagation.tex:99-122`），但 capstone 在 `J_f=0` 的时间无关 Hamilton 量下只检查粒子数。时间阶数通过并不能替代能量漂移检查。

**关闭条件：** 在与 P1-3 相同的有限模式/约化投影约定下推导并实现淬火后 `H_W`，保存整条能量曲线、最大相对漂移和三档步长结果；把阈值加入 `validation`。淬火瞬间允许能量改变，`t>0` 必须按新 Hamilton 量守恒。

### P1-6　总入口允许旧 metrics 冒充本轮产物

`code/run_all.py:79-120` 在子程序退出后只要发现一个可解析且 `all_passed=true` 的指标文件就放行；它没有比较运行前后 mtime/哈希，也不要求指标的生成时刻晚于进程开始。若某脚本因回归而退出0但未重写 metrics，总门禁会误读旧文件。

**关闭条件：** 每项运行前记录或移走旧 metrics，并要求新文件在本次进程启动后产生；更稳妥的做法是写入临时目录后原子替换。汇总中同时记录脚本 SHA-256、metrics SHA-256、schema 版本，并验证每个子检查的布尔值与 `all_passed` 一致。增加一个“脚本退出0但不产出新 metrics 必须失败”的回归测试。

### P1-7　“预注册阈值”措辞超过现有证据

附录C和 README 多次称阈值为“预注册”（`appendices/appC_numerics.tex:66,94-99`；`README.md:39`），但当前证据只证明阈值被写进程序/JSON，不能证明它们在看结果前已由不可变提交或注册记录冻结。

**关闭条件：** 若没有可核验的事前提交/标签，把全书统一改成“预先固定的机器可读阈值”或“显式阈值”；若坚持“预注册”，必须给出运行前的不可变协议提交并在结果中引用其哈希。

### P1-8　图和项目状态不可检索、不可审计

第11--13、15--17章的多幅证据图仍用 `center + includegraphics`，没有 `figure`、caption 或 label，例如 `chapters/part3/ch11_initial_sampling.tex:198-200`、`chapters/part4/ch16_phase_dynamics.tex:167-169`、`chapters/part4/ch17_order_diagnostics.tex:195-197`。读者不能从图目录或正文交叉引用定位它们。与此同时，`MANUSCRIPT_STATUS.md:29-32`、`reviews/full_manuscript_review.md:45-51` 和 `reviews/strict_review_revision_tracker.md:8-23` 仍保留17篇文献、9个程序、4附录、122/126页或大量 `in progress` 等旧状态。

**关闭条件：** 所有证据图改成编号图，caption 至少包含参数/样本量、误差定义、结论与边界，正文用 `\cref` 引用；加入图目录。随后从最终审计数据一次性更新 README、状态文件和 tracker，禁止手工保留互相冲突的数字。

## 对三个核心问题的直接回答

- **知识是否讲解到位？** 核心 TWA 教学目标已经到位：Weyl/Wigner、Moyal 截断、排序、有限模式场、初态抽样、传播、观测量、BdG 淬火和有效性边界形成闭环。开放系统、SOC 与超固态是“边界与方法导论”，不是完成的专著级覆盖，当前正文对此定位基本诚实。
- **论证是否清晰？** 主干论证清晰，尤其“二次体系精确—Kerr 反例—二聚体公平对照—场论实现—应用边界”的证据阶梯很好。剩余最大问题不是叙事，而是 Nambu、`N_W` 和 capstone 投影空间三个跨章约定没有完全闭合。
- **案例是否足够？** 对入门到研究工作流的第一版已经足够，而且均由真实程序生成。capstone 让全书不再只是孤立玩具例；但它必须先关闭 P1-3 至 P1-6，才能承担“端到端可复现证据”的角色。Raman/超固态/开放系统案例可以留待下一版，前提是宣传和章节结论继续限制在现有证据范围内。
