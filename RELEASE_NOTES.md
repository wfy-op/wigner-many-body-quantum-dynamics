# Release notes

## v0.2.0-2026-07-14

相对 `v0.1.0-2026-07-14`，本版完成了严格技术审阅后的系统修订：

- 统一 Weyl 数、正交变量、压缩参数和 BdG/Nambu 约定，补全 Kerr 三阶 Moyal 项、$1/N$ 计数及有限模式投影推导。
- 新增双分量空间场端到端案例，实际覆盖初态抽样、传播、观测量、协方差、粒子数、Weyl 能量以及步长、轨迹数和截止扫描。
- 数值证据扩展为10个程序、72项递归机器检查，并加入陈旧 metrics 防复用门禁和5项回归测试。
- 扩展为18章、6附录、10幅编号证据图、41条已核验参考文献和155页 PDF；加入图目录、文献导读及案例任务地图。
- 全书 LaTeX/BibTeX 警告扫描为零；本版 PDF SHA-256 为 `2EBC06710ECAA8833EF04F36075FDD43182339F9F9DE64CCFCAB8CFA06B9576E`。

边界保持不变：双分量案例是均匀 Rabi 模型，不是 Raman SOC 生产模拟；本版不宣称完成具体体系的平衡超固态判定。
